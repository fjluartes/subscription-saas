# Subscription SaaS
## SubTrack

- Subscription Tracker SaaS app
- Made with Vite + React, Express.js and MongoDB
- Deployment: AWS/Heroku

## Features
- [x] Landing Page
- [x] Subscriptions Dashboard
- [x] Payment Due Date
- [ ] Manage Subscriptions (Pay/Cancel)
- [ ] Active, Past Due, Cancelled status
- [ ] Payments (Stripe)
