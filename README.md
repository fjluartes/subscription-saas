# Subscription SaaS
## SubTrack

- Subscription Tracker SaaS app
- Made with Vite + React, Express.js and MongoDB
- Deployment: AWS/Heroku

## Features
- [x] Landing Page
- [x] Subscriptions Dashboard
- [ ] Manage Subscriptions (Pay or Cancel)
- [ ] Active, Past Due, Cancelled
- [ ] Payment Due Date
- [ ] Payments (Stripe)
